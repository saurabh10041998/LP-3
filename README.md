DES
---
* 56 bit of key.
* 64 bit of plain text.
* initial len(key) = 64 -> 56(0,8,16,32..)
* two main mechanism
	* substitution
	* transposition
* Initial permutation( IP )

 
## S-DES

* 8 bit plain-text/cipher-text
* 10 bits key size of encryption
* two rounds
